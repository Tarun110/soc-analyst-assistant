# MITRE ATT&CK Technique: T1021.001 - Remote Services: Remote Desktop Protocol

**Tactic:** Lateral Movement
**Technique ID:** T1021.001
**Name:** Remote Desktop Protocol

Adversaries may use Valid Accounts to log into a computer using the Remote Desktop Protocol (RDP). The adversary may then perform actions as the logged-on user. RDP is commonly used for lateral movement after initial compromise.

**Detection:**
- RDP connections from unusual source IPs or outside business hours
- RDP from workstation to workstation (unusual lateral pattern)
- Multiple RDP sessions from same account

**Mitigation M1047:** Restrict RDP access via VPN, jump hosts, and network segmentation.
